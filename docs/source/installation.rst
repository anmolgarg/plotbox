
Installation
'''''''''''''''''''''''''''''''''''''''


**Prerequisites**
---------------------------------------

* The usual Python data stack: NumPy, Pandas, and Matplotlib
* Seaborn
* Plotly


**Install with pip or from source (tested on *nix)**
-----------------------------------------------------

with ``pip``

.. code:: sh

	pip install plotbox
    

building from source

.. code:: sh

	git clone https://github.com/anmolgarg/plotbox.git
	cd plotbox
	sudo python setup.py install
